SafeHer – Real-Time Women Safety & SOS Alert System

A geo-enabled women safety web application built using Flask that allows users to send instant SOS alerts along with their live location for emergency assistance.

📌 Project Overview

SafeHer is a real-time women safety system designed to provide quick emergency support.
The application allows users to trigger an SOS alert that shares their live GPS location through a generated Google Maps link.

This system aims to enhance personal safety using web technology and geolocation services.

✨ Features

🚨 One-click SOS emergency alert

📍 Real-time location tracking

🗺 Automatic Google Maps location link generation

🌐 Web-based interface using Flask

🔗 Public access using LocalTunnel

⚡ Fast and lightweight deployment

🛠 Technologies Used

Python

Flask

HTML / CSS

JavaScript (Geolocation API)

Google Maps Integration
